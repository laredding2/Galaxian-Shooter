# Galaxian Shooter

A Python remake of the classic arcade game Galaxian with custom enhancements.

## About

This project recreates the nostalgic space shooter experience of Galaxian, where players defend against waves of alien invaders. The game features both faithful recreation of the original mechanics and additional custom modifications.

## Requirements

- Python 3.x
- Pygame (install via `pip install pygame`)

## How to Play

Run the game with:
```bash
python GalaxianShooter.py
```

- You can also play this game on my website! https://laredding2.github.io/galaxian-shooter.html
  
Control your spaceship to shoot down alien invaders before they reach you!

## Features

- Classic Galaxian gameplay
- Custom game modifications and enhancements
- Built entirely in Python using Pygame
